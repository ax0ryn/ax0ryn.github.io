Introduction

The Mr. Robot CTF on TryHackMe is inspired by the TV show and involves exploiting a vulnerable WordPress installation to gain root access.

Enumeration

Performed an Nmap scan to identify open ports:

nmap -sC -sV -oN nmap_scan.txt 10.10.10.10

Discovered open ports: 80 (HTTP), 443 (HTTPS).

Exploitation

Found a 'robots.txt' file containing a dictionary file 'fsocity.dic' and a key file 'key-1-of-3.txt'. Used the dictionary to brute-force WordPress login credentials.

Privilege Escalation

After gaining access as 'robot' user, found a SUID binary for Nmap. Used Nmap's interactive mode to spawn a root shell:

nmap --interactive
!sh

Conclusion

Successfully captured all three keys and gained root access. This CTF was excellent for practicing web enumeration and privilege escalation techniques.
